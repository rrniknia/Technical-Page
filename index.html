Installing Python

You can check to see if your system has a version of Python installed by using the command python in any command prompt window. If a response from a Python interpreter appears, it will include a version number in the output it displays.

You can install Python by navigating to the Official Python Website and installing the up-to-date version onto your machine.

Variables and Variable Types

In Python and many other programming languages, a variable is a container in memory used to store data values (more accurately, used as a pointer to an object in memory). There are many different types of Python variables which must be used to house the appropriate corresponding data type.

Some basic types of Python variables include:

  • integer - round number (ex. 1, 40, 2000)
  • string - set of characters (ex. "Hello", "going to the mall", "PASSWORD123")
  • float - floating point number/number with a decimal point (ex. 3.14159, 2.5000, 1.9)

To create a variable, simply use the syntax (VARIABLE_NAME = VARIABLE_VALUE). In code, this looks like:

foo = 10
bar = 20

Here, we've created variable foo with a value of 10, and variable bar with a value of 20. The same can be done with strings and floats instead:

random_string = "Hello World!"
floating_point = 1.23456789

After a variable has been created, it can be referenced in the Python prompt and in other operations.

>>>foo
10
>>>bar
20
>>>foo+bar
30
